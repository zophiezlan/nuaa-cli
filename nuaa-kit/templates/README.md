# NUAA Program Kit Templates

This directory contains comprehensive templates for designing, implementing, and managing peer-led harm reduction programs at NUAA.

## Template Catalog (19 Templates)

### Core Program Design Templates

#### 1. **program-design.md** (376 lines)

**Purpose**: Comprehensive program design framework

**Contains**:
- Program overview and evidence base
- Stakeholder journey maps
- Logic model (inputs → activities → outputs → outcomes → impact)
- Risk assessment and mitigation
- Ethical considerations
- Budget estimate
- Impact measurement framework
- Timeline and milestones
- Sustainability planning

**Use for**: New program design, program proposals
**CLI Command**: `nuaa design`

## Template Features

### Comprehensive Coverage

All templates include:

- Clear purpose and objectives
- Step-by-step guidance
- NUAA-specific content (harm reduction, peer-led, consumer participation)
- Practical examples and prompts
- Ethical considerations
- Cultural safety (Aboriginal & Torres Strait Islander, LGBTIQ+, CALD, accessibility)
- Budget sections
- Evaluation/review mechanisms

---

### Placeholder System

Templates use two placeholder systems:

**Bracket Placeholders** (in template content):
- `[Name]` - Program name
- `[Description]` - Target population
- `[Timeframe]` - Duration
- `[Date]` - Current date
- `[Amount]` - Budget amounts
- `[Number]` - Generic numbers

**Token Placeholders** (used in CLI code):
- `{{PROGRAM_NAME}}`
- `{{TARGET_POPULATION}}`
- `{{DURATION}}`
- `{{FUNDER}}`
- `{{AMOUNT}}`
- `{{DATE}}`

---

### Interconnected Templates

Templates reference each other:

- `program-design.md` references `logic-model.md`, `impact-framework.md`
- `proposal.md` can pull from `program-design.md`, `budget-calculator.md`
- `stakeholder-engagement-plan.md` references `partnership-agreement.md`, `communication-plan.md`
- `event-plan.md` references `communication-plan.md`, `risk-register.md`

## Using Templates

### Via CLI (Existing Commands)

```bash
# Generate program design (3 files)
nuaa design "Peer Naloxone Distribution"

# Generate funding proposal
nuaa propose "Peer Naloxone Distribution" --funder "Health NSW" --amount 150000

# Generate impact framework
nuaa measure "Peer Naloxone Distribution"

# Document existing program
nuaa document "Existing Peer Support Program"
```

## Template Principles

**NUAA Values Embedded**:

- **Peer-led**: Templates center lived experience
- **Harm reduction**: Non-judgmental, evidence-based approaches
- **Consumer participation**: $300/session remuneration standard
- **Cultural safety**: Specific considerations for Aboriginal & Torres Strait Islander, LGBTIQ+, CALD communities
- **Accessibility**: Reducing barriers throughout
- **Ethical practice**: Do no harm, informed consent, confidentiality

**Quality Standards**:

- Evidence-based best practices
- Comprehensive yet accessible
- Practical and actionable
- Adaptable to different contexts
- Living documents (designed to evolve)
