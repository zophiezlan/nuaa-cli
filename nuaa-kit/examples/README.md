# NUAA Kit Examples

This directory contains example outputs from NUAA CLI commands to demonstrate what gets generated.

## Available Examples

### Planning & Management

- **[example-stakeholder-engagement.md](example-stakeholder-engagement.md)** - Stakeholder engagement plan (from `/nuaa.engage`)
- **[example-risk-register.md](example-risk-register.md)** - Risk assessment register (from `/nuaa.risk`)

## How to Use These Examples

1. **Browse examples** to see what NUAA CLI generates
2. **Compare with templates** in `../templates/` to understand how placeholders are filled
3. **Run commands yourself** to generate customized outputs for your programs
4. **Use as references** when planning your programs

## Generating Your Own

Examples were generated using commands like:

```bash
nuaa engage "Peer Support Network" "PWUD in Sydney" "12 months"
nuaa risk "Harm Reduction Outreach" "24 months"
nuaa event "Peer Forum 2025" "Forum" "75 people"
nuaa train "Peer Educator Training" "Peer workers" "3 days"
nuaa partner "NSP Program" "Local Health District" "2 years"
```

See [../commands/](../commands/) for complete command documentation.
