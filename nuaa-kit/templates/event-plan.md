## 8. Risk Management

| Risk                        | Mitigation Strategy                                              |
| --------------------------- | ---------------------------------------------------------------- |
| Low attendance              | Promotion, reminders, incentives, confirm with registrants       |
| Speaker cancellation        | Backup speakers identified, flexible program                     |
| AV/tech failure             | Test equipment before, have backup (printed handouts)            |
| Catering issues             | Confirm 48 hours before, have caterer contact on standby         |
| Participant health crisis   | First aid available, naloxone on-site, emergency protocols       |
| Accessibility barriers      | Venue check before, ask about needs in registration, plan ahead  |
| Weather (outdoor component) | Backup indoor option, communicate changes quickly                |
| COVID/health concerns       | Follow current health advice, ventilation, hand sanitizer        |

**Emergency Contacts**:
- Venue: [Phone]
- Caterer: [Phone]
- First aid: [Phone]
- Emergency services: 000

**Incident Response**: [Name] leads, document any incidents, support affected parties
