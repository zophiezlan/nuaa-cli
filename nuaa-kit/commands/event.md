# /nuaa.event - Event Planning Command

**Purpose**: Plan workshops, forums, launches, celebrations, and community events

**Template**: See [event-plan.md](../templates/event-plan.md)

## Command Syntax

```bash
nuaa event <event-name> <event-type> <expected-attendance> [options]
```

## Examples

```bash
nuaa event "Peer Forum Launch" "Forum" "50 people"
nuaa event "Overdose Awareness Day" "Community Event" "100 people"
nuaa event "Training Workshop" "Workshop" "20 people"
```

## What Gets Created

Comprehensive event plan including:
- Event overview and objectives
- Detailed program and schedule
- Logistics (venue, equipment, catering, materials)
- Promotion and registration strategy
- Speakers and presenters coordination
- Event team roles and responsibilities
- Participant support and accessibility
- Risk management and contingencies
- Budget and resources
- Evaluation plan
- Post-event follow-up
- Comprehensive planning checklist

## Event Types Supported

- Workshops and training
- Forums and consultations
- Program launches
- Celebrations (IDPW, Overdose Awareness Day)
- Conferences
- Community gatherings
- Advocacy events

## Best Practices

1. **Plan early**: 8 weeks minimum for major events
2. **Budget realistically**: Include hidden costs (parking, AV, etc.)
3. **Make it accessible**: Venue, timing, catering, childcare
4. **Build in contingencies**: Weather backup, speaker cancellation plans
5. **Promote widely**: Multiple channels, start 4 weeks before
6. **Provide support**: Transport, honorariums, accessible formats
7. **Follow up**: Thank yous, photos, evaluation within 1 week
