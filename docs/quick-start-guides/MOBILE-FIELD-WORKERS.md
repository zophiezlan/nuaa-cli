## Choose Your Method

### Option 1: Text Message (Fastest - 30 seconds)

**For**: Quick updates, distribution logs, outreach check-ins

**How**:

1. Open your text messages
2. Text to: `[number]`
3. Write your update
4. Send!

**Example**:

```
Text to: [number]

OUTREACH Kings Cross
20 people 45 needles
All good
```

**You get back**:

```
‚úÖ Report saved!
View: [link]
Need resupply? Reply YES/NO
```

**Format**:

```
[TYPE] [Location]
[Numbers and supplies]
[Any notes]
```

**Types**:

- OUTREACH
- DISTRIBUTE (peer distribution)
- FESTIVAL
- RESUPPLY (request supplies)

---

### Option 2: Mobile Web (5 minutes)

**For**: Full reports with more details

**How**:

1. Open your phone's web browser
2. Go to: `nuaa.org.au/tools` (or bookmark it!)
3. Tap your team name
4. Fill in the form
5. Tap Submit

**Works on**:
‚úì iPhone (Safari)
‚úì Android (Chrome)
‚úì Any smartphone
‚úì Old phones too!

**Features**:

- Large buttons (easy to tap)
- Can use voice typing
- Auto-saves as you go
- Works offline (submits when you get signal)
- Can add photos

---

### Option 3: Email from Phone

**For**: When you have a few minutes

**How**:

1. Open your email app
2. Email to: `nuaa@nuaa.org.au`
3. Subject: Report type
4. Answer the questions
5. Done!

See "Email-Only Users" guide for details.

## SMS Quick Reference

### Outreach Update

```
OUTREACH [Location]
[# people] [# needles]
[Notes]
```

Example:

```
OUTREACH Redfern
15 people 30 needles
Good session
```

---

### Distribution Log

```
DISTRIBUTE
[# people] [# needles]
[Resupply YES/NO]
```

Example:

```
DISTRIBUTE
8 people 25 needles
Resupply YES
```

---

### Festival/Event

```
FESTIVAL [Event Name]
[# contacts] [Notes]
[Urgent YES/NO]
```

Example:

```
FESTIVAL Lost Paradise
50 contacts 2 medical referrals
Urgent NO
```

---

### Resupply Request

```
RESUPPLY
[What you need]
[When needed]
```

Example:

```
RESUPPLY
Needles 1ml and 3ml
By Friday
```

## Mobile Web Tips

### On Your Phone

**Big Buttons**: All buttons are large and easy to tap

1. Open `nuaa.org.au/tools` in Chrome
2. Tap menu (three dots)
3. Tap "Add to Home Screen"
4. Now it's an icon on your phone!

**Benefit**: Opens like an app, easier to find

## Troubleshooting

### "Web page won't load"

- Check you have signal/WiFi
- Try closing and reopening browser
- Clear browser cache
- Try different browser

### "SMS not sending"

- Check you have the right number
- Check your phone credit/plan
- Try regular text first to test
- Use email or web instead

### "Form won't submit"

- Check all required fields filled
- Try again in 1 minute
- Check internet connection
- Save draft and try later

### "Lost my draft"

- Web saves automatically
- Check browser's "Recently Closed"
- Contact tech support - might be recoverable
