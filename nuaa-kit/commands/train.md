# /nuaa.train - Training Curriculum Command

**Purpose**: Design comprehensive training curricula for peer workers, volunteers, or staff

**Template**: See [training-curriculum.md](../templates/training-curriculum.md)

## Command Syntax

```bash
nuaa train <training-name> <target-audience> <duration> [options]
```

## Examples

```bash
nuaa train "Peer Worker Training" "New peer workers" "2 days"
nuaa train "Naloxone Training" "Community members" "3 hours"
nuaa train "Volunteer Induction" "NUAA volunteers" "4 weeks"
```

## What Gets Created

Complete training curriculum including:
- Training overview and learning objectives
- Pedagogical principles (experiential, peer-led, harm reduction)
- Detailed training schedule
- 8+ training modules with activities
- Facilitation guidance and tips
- Training materials list (participant & facilitator)
- Participant support and accessibility
- Assessment and evaluation
- Budget and resources

## Training Modules Include

1. Introduction and welcomes
2. Core content modules (customizable)
3. Skills practice and activities
4. Self-care and sustainability
5. Evaluation and next steps

## Use Cases

- Peer worker training and orientation
- Volunteer program induction
- Harm reduction education workshops
- Skills development programs
- Partner staff capacity building

## Best Practices

1. **Co-design with peers**: Lived experience input is essential
2. **Mix methods**: Vary between presentation, activities, discussion
3. **Build in breaks**: Respect participants' capacity
4. **Provide support**: Accessible venue, catering, transport
5. **Remunerate fairly**: $300/session for peer facilitators
6. **Evaluate and iterate**: Improve based on participant feedback
